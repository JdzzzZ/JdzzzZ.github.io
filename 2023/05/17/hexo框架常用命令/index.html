hexo框架常用命令

init

新建一个网站

  

默认在目前文件夹建立网站

new

新建一篇文章

generate

生成静态文件

publish

发表草稿

server

启动服务器

deploy

部署网站

render

渲染文件

clean

清除缓存文件和已生成的静态文件

在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令。
